# MeerChat

Chatting Shell with Golang

## 하나 목표

각자 *브랜치*를 파서 작동하는 채팅(서버) 구현하기.

- 다음모임때 코드 리뷰를 해서 채팅서버 통합하기.
- P2P던, 채팅방만들기던 메세지를 주고받을 수 있어야 함.
- 터미널이던 웹이던 뭐든 작동해야함.
